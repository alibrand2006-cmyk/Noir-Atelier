# ✨ Luxury Perfume eCommerce Website

🚀 **Live Demo:** https://alitaskai.lovable.app/

An elegant and high-converting perfume eCommerce platform crafted to deliver a premium online fragrance experience. This project blends modern UI/UX design with smooth performance and visually rich content to simulate a real luxury brand.

---

## 🌟 Features

* 📱 Fully responsive (mobile-first design)
* 🎨 Premium dark luxury UI with modern aesthetics
* 🖼 Unique high-quality product visuals (no repetition)
* 🔍 Advanced product filtering and search
* 🧴 Detailed product pages with fragrance notes (top, middle, base)
* 🛒 Add to cart and smooth checkout flow
* 👤 User authentication (login/signup)
* ⚡ Fast performance with clean UI interactions

---

## 🛠 Tech Stack

* **Frontend:** Next.js / HTML / CSS / JavaScript
* **Styling:** Tailwind CSS
* **Deployment:** Vercel
* **AI Tools:** ChatGPT / Cursor
